IDEA 快捷键整理

发表于 2018-03-16 | 分类于 编辑器 | | 阅读次数:
字数统计: 450 | 阅读时长 ≈ 2

IntelliJ IDEA 的便捷操作性，快捷键的功劳占了一大半，对于各个快捷键组合请认真对待。IntelliJ IDEA 本身的设计思维是提倡键盘优先于鼠标的，所以各种快捷键组合层出不穷，对于快捷键设置也有各种支持，对于其他 IDE 的快捷键组合也有预设模板进行支持。

Cmd + F 在当前文件进行文本查找
Cmd + shift + F 提供全局匹配查找等功能
Cmd + R 在当前文本进行替换
Cmd + Alt + F7 Show usages
Cmd + F12 展示当前类结构，所有方法及属性等等
option + Cmd + O 查找函数调用
shift + option + Cmd + U 展示当前类（当前光标指针位置）所有的继承依赖关系
Cmd + E 最近打开以及更改的文件列表
shift + Cmd + E 只展示修改过的文件
Ctrl + option + O 优化导入包
Ctrl + option + L 格式化代码
Cmd + Shift + U 大小写转换
fn + Cmd + 左右键 跳转到文件头部或尾部
Ctrl + option + N 新建文件，类名前加上xxxPackage.XxxClass；这个xxxPackage就是包名{可以不存在})
Ctrl + H 查看当前类的继承结构-默认是子类
Cmd + U 跳转到父类
Control + J 查看方法注释
shift + Cmd + 上下键 移动当前选中代码到上一行或下一行
Cmd + D 复制当前代码到下一行
Cmd + delete 删除当前行
Cmd + N 生成get、set以及构造器
Cmd + shift + O 文件查找（重复按一次O可以寻找classpath中的资源文件）
Cmd + O 类文件查找（重复按一次O可以寻找当前模块&当前项目的类文件）
Ctrl + O 重写父类方法
Ctrl + I 实现接口
option + Cmd + T 代码环绕，生成try、catch之类
